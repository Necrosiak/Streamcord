# Steamcord

**Discord в игровом режиме Steam** — плагин [Decky Loader](https://github.com/SteamDeckHomebrew/decky-loader) для Steam Deck / Bazzite / SteamOS.

## Как это работает

Steamcord запускает **[Vesktop](https://github.com/Vencord/Vesktop)** — настоящий нативный клиент Discord — невидимо в фоне и управляет им через Chrome DevTools Protocol. Плагин внедряет в него небольшой клиент и выводит всё в **меню быстрого доступа** Steam.

Переход на нативность решает сложные проблемы старого подхода со скрытым браузером: **ваш микрофон и голосовой звук работают нативно**, точно как в десктопном приложении Discord — без трюков с захватом, без обходов автовоспроизведения. Vesktop запускается (и устанавливается при отсутствии) автоматически, остаётся в системе после перезагрузки и никогда не требует окна рабочего стола в игровом режиме.

## Возможности

- **Вход по QR-коду** — Отсканируйте QR-код приложением Discord на телефоне, чтобы войти мгновенно. На телефоне: *Discord → Настройки → Сканировать QR-код*, затем наведите на код в панели. Без ввода пароля на Deck.
- **Полноэкранный вход (запасной)** — Открывает Discord на весь экран для входа по почте/паролю или решения CAPTCHA, когда QR недоступен.
- **Голосовой чат** — Заходите в голосовые каналы и слышьте всех; каждый участник показан в реальном времени (кольцо речи, значки заглушения/оглушения) и ползунок громкости на каждого (0–200 %). Микрофон и звук нативные (Vesktop).
- **Личные сообщения (ЛС и групповые)** — Просматривайте беседы и начинайте/присоединяйтесь к голосовым звонкам с друзьями прямо из меню быстрого доступа. Активные звонки выделяются.
- **Голосовой обозреватель серверов** — Смотрите, в каких голосовых каналах есть люди (с аватарами), прежде чем зайти.
- **Текстовые каналы** — Читайте последние сообщения канала сервера из QAM и отвечайте с помощью экранной клавиатуры Steam (она открывается сама при фокусе на поле ввода).
- **Статус Discord** — Задавайте свой статус (в сети / не на месте / не беспокоить / невидимый) из QAM. Необязательная автосинхронизация заставляет Discord **следовать вашему статусу Steam** в фоне; выбор статуса вручную возвращает в ручной режим.
- **Заглушить / Оглушить / Отключиться** — Голосовое управление одним касанием из QAM.
- **Go Live (демонстрация экрана)** — Делитесь всем экраном в голосовом канале.
- **Уведомления в игре** — ЛС и упоминания появляются как уведомления Steam (и учитывают ваш статус Discord — отключены при невидимом / не беспокоить).
- **Push-to-talk** — С физической кнопкой (по умолчанию R5).
- **Отправка скриншотов** — Отправьте скриншот Steam в любой канал Discord.
- **[Vencord](https://vencord.dev/)** встроен в Vesktop, давая доступ к его экосистеме плагинов.

## Установка

> **Пока нет в Decky Store.** Ручная установка через режим разработчика.

1. Включите **режим разработчика** в Decky → Общие настройки
2. Перейдите в **Разработчик** в настройках Decky
3. Установите по URL:
   `https://github.com/Necrosiak/Steamcord/releases/latest/download/Steamcord.zip`

Vesktop устанавливается и запускается плагином автоматически при первом запуске. Достаточно войти один раз (QR или полный экран) — и вы останетесь в системе.

### Требование (демонстрация экрана)
Демонстрация экрана работает сразу — плагин автоматически устанавливает зависимость Python (aiohttp) при первом запуске. GStreamer берётся из системы.

## Сборка из исходников

```bash
git clone https://github.com/Necrosiak/Steamcord
cd Steamcord
pnpm install
pnpm run build
# скопируйте dist/, main.py, defaults/, plugin.json, package.json в ~/homebrew/plugins/Steamcord/
sudo systemctl restart plugin_loader
```
